# Changelog

## v2025.01 — Initial release

- `data/counties.csv` — 21 Swedish counties with codes, full names, and short names
- `data/municipalities.csv` — 290 Swedish municipalities with county linkage and short names
- `data/municipality_county.csv` — Denormalized municipality-county join table (290 rows)
- `data/postal_to_municipality.csv` — Postal code to municipality mapping (15,463 postal codes)
- `scripts/validate.py` — Validation script (Python 3 stdlib, no dependencies)
- `schemas/column_descriptions.md` — Detailed column documentation
- `README.md` — Usage examples, data sources, and format documentation
